# Bartender Sales Intelligence Dashboard

## Overview
This project analyzes simulated bar sales data using Python to uncover revenue trends, top-performing drinks, and peak operating periods. The analysis is designed to show how transactional data can support staffing, inventory, and sales decisions.

## Tools Used
- Python
- pandas
- matplotlib
- Google Colab

## Dataset
This project uses a simulated transactional dataset representing bar sales. The data includes:
- date and time of transaction
- item name
- drink category
- quantity sold
- price
- payment type

## Project Goals
- Identify the drinks generating the most revenue
- Analyze hourly and shift-based sales patterns
- Evaluate category performance
- Provide business recommendations based on observed trends

## Key Insights
- Old Fashioneds and Whiskey Sours were the top revenue-generating drinks.
- Revenue peaked during evening and late-night hours.
- The Evening shift produced the largest share of total revenue.
- Lower daytime demand suggests opportunities for promotions and staffing optimization.

## Business Recommendations
- Staff more heavily during evening and late-night shifts.
- Stock more inventory for top-selling premium cocktails.
- Introduce promotions during slower hours.
- Optimize scheduling around demand by shift.
